# Repositories

GateKey is split across several repositories:

| Repository | Description |
|------------|-------------|
| [GateKey](https://github.com/dye-tech/GateKey) | Core server, CLI client, and web UI |
| [gatekey-helm-chart](https://github.com/dye-tech/gatekey-helm-chart) | Helm chart for Kubernetes deployment |
| [gatekey-kustomize](https://github.com/dye-tech/gatekey-kustomize) | Kustomize manifests for Kubernetes |
| [gatekey-android-client](https://github.com/dye-tech/gatekey-android-client) | Android mobile client |

## Contributing

1. Fork the relevant repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request

See each repository's README for specific build and development instructions.
